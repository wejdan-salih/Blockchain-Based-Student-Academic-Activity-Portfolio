# Blockchain-Based Student Academic Activity Portfolio

## 📌 Overview
This project is a blockchain-based academic portfolio system designed to provide secure, fast, and tamper-proof verification of student credentials.

It enables universities, students, and employers to verify academic records instantly using blockchain technology.

---

## 🎯 Objectives
- Ensure academic records are tamper-proof
- Enable instant credential verification
- Give students ownership of their academic data
- Reduce fraud and manual verification processes

---

## ⚙️ Technologies Used
- Solidity (Smart Contracts)
- Ethereum (EVM)
- IPFS (Decentralized Storage)
- Next.js (Frontend Framework)
- Ethers.js (Blockchain Interaction)
- MetaMask (Authentication)
- Hardhat (Development Environment)
- MySQL (Off-chain Database)

---

## 🏗️ System Architecture
1. Academic data is uploaded and stored on IPFS  
2. IPFS hash is stored on the Ethereum blockchain  
3. Smart contracts manage verification  
4. Users can instantly verify credentials via blockchain  

---

## 🚀 Features
- 🔐 Secure and tamper-proof records  
- ⚡ Fast verification (under 2 seconds)  
- 🎓 Student-owned academic portfolio  
- 🌐 Decentralized storage using IPFS  
- 🧾 Transparent verification system  

---

## 📊 Project Impact
- Eliminates fake academic certificates  
- Reduces verification time significantly  
- Improves trust between institutions and employers
